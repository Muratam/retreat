        - y: 重力がある
        - z: 後ろは見えない
        - w: 一方通行(避けることもできない) / 変形できる。Enum的。別軸の方がいいかも
      - CX) 椅子の後ろ: 自分にとって？相手にとって？曖昧。

      - ~の: of (tupleシンボル生成？)
      - ~として: as
      - 〜なし/〜ありで: with / without
    - 目的の概念
      - ~のために: for
    - 数値
